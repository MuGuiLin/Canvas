Canvas标签必须和js脚本配合才能用。Cancas.getContext("2d")//绘图对象
我是DIV(理解坐标)

canvas元素本身是没有绘图能力的。所有的绘制工作必须在 JavaScript 内部完成：

.getContext("2d");//对象是内建的 HTML5 对象

.fillStyle = "blue"; //设置背景色

.strokeStyle = "white";//线条颜色

.fillRect(200,60,200,100); // 绘制矩形： 左，上，宽，高规定了形状、位置和尺寸。

.stroke();//stroke() 方法会实际地绘制出通过 moveTo() 和 lineTo() 方法定义的路径。默认颜色是黑色。

CV.beginPath();//结束或丢弃任何当前定义的路径并且开始一条新的路径。它把当前的点设置为 (0,0)。

.arc(300,350,150,0,Math.PI*2,true);//绘制圆形

.closePath(); //从当前点到开始点的路径.(返回合并)

.fill();//fill() 方法填充当前的图像（路径）。默认颜色是黑色。
